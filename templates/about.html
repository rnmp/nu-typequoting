About the project

{% for image in images %} {% endfor %}

Typequoting brings storytelling and typography together in a particular way.

Every visitor of the site is asked to create a story inspired by five random elements in less than a minute. These elements are directly related to the shape and resemblance of letterforms. The point of the game is to make connections between elements and end up with a unique story.

This website actually started as a poster project for William Bevington's Typography class on Parsons School of Design, where classmates were asked to write stories based on compositions inspired by typographic elements.

All the stories are publicly available on the submissions page. If you would like to add more images to the collection featured above, please send an email to rolando@bandd.co.
